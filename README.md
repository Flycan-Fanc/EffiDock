# EffiDock（建设中...）

**EffiDock** 是一个支持模块安装 / 卸载的 AI 增强型生产力工作台。  
用户可以按需启用或关闭不同工具模块，逐步搭建属于自己的效率系统，例如 Todo、日记、灵感、财务、项目管理等。

> 一个轻量、清爽、可扩展的生产力平台，面向个人与团队场景。

## ✨ 项目愿景

很多效率工具要么功能过于单一，要么做成了臃肿的“大而全”产品。  
EffiDock 想做的是一个 **模块化生产力平台**：用户不需要被迫接受一整套固定功能，而是可以像使用“应用商店”一样，只保留自己真正需要的工具。

长期来看，我希望 EffiDock 能成为一个类似 **“效率工具工作台”** 的产品：

- 需要的模块就安装
- 不需要的模块就卸载
- 保持界面清爽、功能聚焦
- 通过 AI 辅助提升计划、执行与复盘效率

## 🧩 核心理念

EffiDock 围绕三个核心理念展开：

### 1. 模块化

每一种工具都被视为独立模块。  
用户可以自由启用、停用或移除模块，而不是被固定功能绑定。

### 2. AI 增强

AI 在 EffiDock 中不只是聊天入口，而是一个真正参与工作流的辅助能力。  
它可以帮助用户：

- 分析计划是否合理
- 识别执行中的问题
- 给出优化建议
- 总结进度与复盘内容

### 3. 多端扩展

EffiDock 的长期目标是支持多端使用：

- **MVP**：React + TypeScript + Web
- **桌面端**：React + Tauri
- **移动端**：先做响应式适配，后续再决定是否开发独立 App

## 🚀 计划中的功能模块

项目会按阶段逐步实现。  
目前规划中的模块包括：

### 个人效率模块

- Todo / 待办清单
- 日记
- 灵感记录
- 标签系统
- 日历 / 计划视图

### 财务管理模块

- 个人账单记录
- 预算管理
- 收支分析
- 消费趋势统计

### 项目管理模块

- 任务拆分
- 工期管理
- 时间线 / 甘特图
- 里程碑管理

### 协作模块

- 团队空间
- 任务指派
- 评论 / 讨论
- 权限与角色管理

### AI 功能模块

- 计划合理性分析
- 执行情况复盘
- 每周 / 每月总结
- 项目延期风险提示
- 财务趋势分析

## 🎨 界面风格方向

EffiDock 将采用 **简约、清爽、克制** 的界面风格：

- 结构清晰
- 信息层级明确
- 尽量减少视觉噪音
- 低饱和配色
- 强调产品感而不是装饰感

## 🗺 开发路线

### Phase 0

- 明确产品定位
- 确定 MVP 范围
- 设计模块结构
- 搭建初始 UI 体系

### Phase 1

- 完成 Web 端 MVP
- 实现平台壳子
- 完成 Todo / Diary / Inspiration
- 接入 AI 计划建议

- 使用 Tauri 扩展桌面端
- 增强本地持久化能力
- 支持文件导入 / 导出
- 优化工作台体验

### Phase 3

- 增加财务与复盘模块

### Phase 4

- 增加项目管理模块

### Phase 5

- 增加协作与云端同步能力

### Phase 6

- 完成真正的插件 / 应用商店式模块系统
